## Опис

NEFK Tweaker — це набір оптимізаційних налаштувань та утиліт, реалізований на **PowerShell**, що прискорює завантаження системи та покращує продуктивність Windows 10 і Windows 11. Скрипт містить опції для відключення небажаних служб, очищення тимчасових файлів, налаштувань приватності та інших оптимізацій, спрямованих на зменшення навантаження системи.

Проєкт розгорнуто на платформі **Vercel** і доступний для швидкого запуску через просту команду (див. розділ *Використання*).

---

## Особливості

* Оптимізація завантаження системи (вимкнення/налаштування служб та задач)
* Очищення тимчасових файлів та кешу
* Налаштування параметрів приватності Windows
* Мінімалістичний інтерфейс запуску через PowerShell
* Можливість подальшого розширення та налаштування під конкретні потреби

---

## Передумови

* ОС: Windows 10 або Windows 11
* Права адміністратора для виконання більшості команд скрипта
* PowerShell (версія, що постачається разом із ОС)
* Зовнішній доступ до інтернету для завантаження скрипта з Vercel (якщо використовуєте команду `irm ... | iex`)

## Встановлення

> NEFK Tweaker не вимагає класичного встановлення — скрипт можна запустити напряму з хостингу.

### Швидкий запуск (рекомендовано для досвідчених користувачів)

У PowerShell (з правами адміністратора) виконайте:

```powershell
irm https://nefktweak.vercel.app/win | iex
```

> **Увага:** команда `irm ... | iex` завантажує та виконує скрипт безпосередньо з інтернету. Виконуйте її тільки якщо довіряєте джерелу та розумієте зміст скрипта.

---

## Рекомендований робочий процес (безпечніший підхід)

1. Скачайте скрипт локально та перегляньте його вміст перед виконанням:

```powershell
Invoke-RestMethod https://nefktweak.vercel.app/win -OutFile .\nefktweak.ps1
notepad .\nefktweak.ps1
```

2. Після перевірки — запустіть локально:

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\nefktweak.ps1
```
